Documentation

Please read the documentation below on how to use the XYZ Dictionary free API. This public API should only be used for testing, personal or infrequent use.

Purpose of the API: Get word definitions

How to use: The basic syntax of a URL request to the API is shown below:

https://api.result?word=[query param value]

As an example, to get definition of English word code, you can send request to:

https://api.result?word=code
